install applications quickly and easily with okutagon

to get started, simply select the applications you want to install from the list provided. recommended apps are indicated in bold and red.

once you've made your selections, click the "generate" button, and our tool will generate a chocolatey-based script code that you can use to install all the applications you've chosen.

the generated script will be copied automatically to your clipboard. simply paste the script code into your terminal and press enter. it's that easy!

note: in order to use this tool, javascript must be enabled in your browser. if you are experiencing issues, please make sure to check your browser settings and enable javascript before trying again.
