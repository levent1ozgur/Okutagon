okutagon is a tool that designed to help you quickly and easily install applications on your pc, for example right after formatting the pc.this tool uses the package manager called chocolatey for windows and homebrew for macos. with just a few clicks, you can select the applications you want to install and generate a script code that will automatically install them all at once.
to get started, simply select the applications you want to install from the list provided. recommended apps is indicated in bold and red marked.
once you've made your selections, click the "generate" button, and our tool will generate a script code that you can use to install all the applications you've chosen.
generated script will be copied automaticly the clipboard, simply paste the generated script code into your powershell in administrator mode and press enter. it's that easy!
reminder: please note that in order to use this tool, javascript must be enabled in your browser. if you are experiencing issues, please make sure to check your browser settings and enable javascript before trying again.
