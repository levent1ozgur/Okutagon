install applications quickly and easily with okutagon's multi installer

to get started, simply select the applications you want to install from the list provided. recommended apps are indicated in bold and white marked.

while you are selecting options, a homebrew-based script will be dynamically created at the bottom of the page that you can use to install all the applications you've chosen.

after you finish making your selections, copy the code paste it into powershell and press enter. it's that easy!

note: in order to use this tool, javascript must be enabled in your browser. if you are experiencing issues, please make sure to check your browser settings and enable javascript before trying again.
